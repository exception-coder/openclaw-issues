# OpenClaw Knowledge

这里是 OpenClaw 相关知识沉淀的父目录，统一收纳两类内容：

- [`issues/`](./issues/)：故障、排查、修复、复盘
- [`capabilities/`](./capabilities/)：已经形成并验证过的能力、方法、实战案例

## 设计目的

把“问题记录”和“能力沉淀”放到同一个知识域下，但保持各自边界：

- `issues` 回答：出了什么问题，怎么查，怎么修
- `capabilities` 回答：我现在会什么，在哪些场景已经做成过
